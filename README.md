# Обработка, анализ и визуализация данных об успеваемости студентов с использованием `pandas` и `matplotlib`

Шаблон репозитория для сдачи лабораторной работы в рамках дисциплины "Программирование на Python"

## Структура репозитория

Ваш репозиторий должен был оформлен по следующей структуре:

```text
.
├── data/
│   └── StudentsPerformance.csv
├── lab.ipynb
├── README.md
├── requirements.txt
└── .sourcecraft/
    └── ci.yaml
```

### Назначение файлов

- `data/StudentsPerformance.csv` — исходный датасет;
- `lab.ipynb` — основной notebook с выполненной лабораторной работой;
- `README.md` — краткое описание проекта;
- `requirements.txt` — список используемых библиотек;
- `.sourcecraft/ci.yaml` — конфигурация CI/CD pipeline.


## Требования к оформлению notebook

Ваш notebook должен быть оформлен так, чтобы его можно было понять без устных пояснений.

Чеклист для самопроверки:

- все ячейки должны выполняться без ошибок;
- код должен быть расположен по шагам;
- у каждого логического блока должен быть заголовок;
- у каждого графика должны быть:
  - заголовок;
  - подпись оси X;
  - подпись оси Y;
- под каждым графиком должен быть краткий текстовый вывод;
- в конце должен быть общий вывод по работе.

## Требования к CI/CD

В репозитории должен быть настроен CI/CD пайплайн, который:

- запускается автоматически;
- устанавливает зависимости;
- выполняет notebook;
- генерирует отчет;
- сохраняет отчет как артефакт (в формате HTML)

## Материалы для сдачи

Для сдачи лабораторной работы необходимо предоставить:

- ссылку на репозиторий;
- заполненный notebook;
- рабочий pipeline;
- готовый HTML-отчет в artifacts.
